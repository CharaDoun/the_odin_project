# Project Odin Recipes

## Description
This is my version of the html capstone project of the Odin Project. The goal is to create a website that contains three of my favorite recipes all of them linked to a main webpage by using basic html.

## Learning Outcomes
- Create a website containing three recipes
- Get familiar with proper usage of html tags
- Create proper file structure of a website
- Crete proper linking between different html files
- Using git and doing proper commits with subjects and bodies where needed
